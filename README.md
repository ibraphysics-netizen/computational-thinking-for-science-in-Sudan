# Computational Thinking for Science Students

Welcome to **Computational Thinking for Science Students**, a **free, low-resource educational initiative** designed to empower young scientists in Sudan and around the world.

This course introduces **foundational computational and analytical skills** that are essential for modern science, including physics, chemistry, materials science, and engineering. You do **not need prior programming experience**, and the course is designed to work with minimal internet or computer resources.
